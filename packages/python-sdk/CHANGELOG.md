# cavendo Python SDK Changelog

## 0.1.0 (2026-02-18)

Initial release.

- Typed client for the Cavendo Engine REST API
- Task operations: list, get, claim, update status, submit deliverables
- Deliverable operations: submit with files, actions, and summary
- Knowledge base access
- Authentication via agent keys (`cav_ak_`) and user keys (`cav_uk_`)
- Context gathering for task execution
